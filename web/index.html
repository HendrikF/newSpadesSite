NewSpades is a first person shooter (FPS) inspired by Ace of Spades.

The Story

In the beta phase Ace of Spades was free to play and made a lot of fun. The 3rd party game server pyspades was written in Python and C++ and easily extensible, which means that many game modes were implemented. Now, after the beta phase, AoS is offered on Steam and graphically modernized.

To us the old Beta-AoS has many advantages. One advantage is the ability to host community servers.

Our Plan

Since the client could be better than it currently is and the code is not available, we want to create a similar game. The focus should be on flexibility and community. Each server should give a completely unique experience. We want to reach this by making the server able to adapt all parameters of a game environment:

  • Number of teams
  • Customize the available weapons

This should be possible with a simple and flexible scripting system on the server.

Some Ideas

  • Open source
  • As community-driven as possible
  • Free to play
  • Client does not need to be modified, server controls everything
  • User registration online possible, not necessary
  • User authentication (like online-mode in Minecraft)
  • Leaderboards
  • Ban lists
  • Everyone can run a server
  • Global server list
  • Server modifications (Scripts) should be written by the community
